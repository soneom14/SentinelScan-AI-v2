# SentinelScan-AI v2

A modular Python-based network reconnaissance and vulnerability assessment tool developed for cybersecurity learning and security auditing.

## Features

* TCP Port Scanning
* Service Detection
* Colorized Terminal Output
* Automated Report Generation
* Modular Architecture
* GitHub Version Control

## Project Structure

```text
SentinelScan-AI-v2
│
├── modules
│   ├── port_scanner.py
│   ├── report_generator.py
│   └── __init__.py
│
├── reports
│
├── main.py
├── requirements.txt
├── README.md
└── LICENSE
```
